# NetproTaskApp
大学の講義の最終課題として制作したもので，友人と2人で制作しました。<br>
tomcat+jsp+servletで動作するタスク管理ツールです。
 
## 動作環境について
サーバ：Tomcat9（Java11）<br>
データベース:PostgreSQL 12<br>

## DEMO
herokuでデモを設置しています。恐らくすぐ消します。<br>
